## üë§ User Guide

### Registration and Login

#### Creating an Account
1. Click the "Sign Up" button on the login page
2. Fill in the registration form:
   - **Username**: Choose a unique username (minimum 3 characters)
   - **Email**: Enter a valid email address
   - **Password**: Create a secure password (minimum 6 characters)
   - **Confirm Password**: Re-enter your password
3. Click "Sign Up" to create your account
4. You'll be redirected to the login page

#### Logging In
1. Enter your email address and password
2. Click "Login" to access your account
3. You'll be redirected to the quiz dashboard

### Taking Quizzes

#### Browsing Quizzes
1. On the main dashboard, you'll see all available quizzes
2. Use the technology filter to find quizzes by topic:
   - JavaScript
   - React
   - Node.js
   - Python
   - And more...
3. Each quiz card shows:
   - Quiz title
   - Technology category
   - Creation date

#### Starting a Quiz
1. Click on any quiz card to start taking the quiz
2. Read each question carefully
3. Select your answer from the multiple choice options
4. Use "Previous" and "Next" buttons to navigate between questions
5. Click "Submit Quiz" when you're ready to finish

#### Quiz Results
After submitting a quiz, you'll see:
- Your total score (percentage)
- Number of correct answers
- Total number of questions
- Performance feedback

### Viewing Your History

#### Accessing Quiz History
1. Click "My Attempts" on the main dashboard
2. View all your past quiz attempts
3. See your scores and performance over time

#### Understanding Your Results
- **Score**: Percentage of correct answers
- **Correct Count**: Number of questions answered correctly
- **Total**: Total number of questions in the quiz
- **Date**: When you took the quiz

## üë®‚Äçüíº Admin Guide

### Admin Dashboard

#### Accessing Admin Features
1. Login with an admin account
2. Click "Admin Panel" in the header
3. Access all quiz management features

### Quiz Management

#### Creating a New Quiz
1. Click "Create New Quiz" in the admin panel
2. Fill in the quiz details:
   - **Title**: Enter a descriptive quiz title
   - **Technology**: Select the technology category
   - **Published**: Toggle to make quiz visible to users
3. Add questions:
   - **Question Text**: Enter the question
   - **Options**: Add multiple choice options (minimum 2)
   - **Correct Answer**: Select which option is correct
   - **Add Question**: Click to add more questions
4. Click "Save Quiz" to create the quiz

#### Editing Existing Quizzes
1. In the admin panel, find the quiz you want to edit
2. Click "Edit" next to the quiz
3. Modify any quiz details or questions
4. Click "Update Quiz" to save changes

#### Publishing Quizzes
1. When creating or editing a quiz, toggle the "Published" option
2. Published quizzes are visible to all users
3. Unpublished quizzes are only visible to admins

#### Deleting Quizzes
1. Find the quiz you want to delete
2. Click "Delete" next to the quiz
3. Confirm the deletion
4. **Warning**: This action cannot be undone

### Quiz Analytics

#### Viewing Quiz Performance
1. Access the admin dashboard
2. View quiz statistics:
   - Number of attempts
   - Average scores
   - Popular quizzes
   - User engagement

## üîß Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Cannot login with correct credentials
**Solutions**:
- Check if Caps Lock is on
- Verify email address is correct
- Try resetting your password
- Clear browser cache and cookies

**Issue**: "Invalid token" error
**Solutions**:
- Logout and login again
- Clear browser storage
- Check if your session has expired

#### Quiz Taking Issues
**Issue**: Quiz not loading
**Solutions**:
- Refresh the page
- Check your internet connection
- Try a different browser
- Clear browser cache

**Issue**: Cannot submit quiz
**Solutions**:
- Ensure all questions are answered
- Check your internet connection
- Try refreshing the page
- Contact support if problem persists

#### Performance Issues
**Issue**: Slow loading times
**Solutions**:
- Check your internet connection
- Close other browser tabs
- Clear browser cache
- Try a different browser

### Browser Compatibility

#### Supported Browsers
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

#### Mobile Devices
- iOS Safari 14+
- Chrome Mobile 90+
- Samsung Internet 13+

### Error Messages

#### Common Error Messages
- **"Email already exists"**: Try a different email address
- **"Invalid email format"**: Check your email address format
- **"Password too short"**: Use at least 6 characters
- **"Quiz not found"**: The quiz may have been deleted
- **"Access denied"**: You don't have permission for this action

## ‚ùì FAQ

### General Questions

**Q: Can I retake a quiz?**
A: Yes, you can take any quiz multiple times. Each attempt is recorded separately.

**Q: How are scores calculated?**
A: Scores are calculated as (correct answers / total questions) √ó 100.

**Q: Can I see the correct answers after taking a quiz?**
A: Currently, the app shows your score but not the correct answers. This feature may be added in future updates.

**Q: Is my data secure?**
A: Yes, all data is encrypted and stored securely. Passwords are hashed and never stored in plain text.

### Technical Questions

**Q: What if I lose my internet connection during a quiz?**
A: Your progress may be lost. It's recommended to have a stable internet connection while taking quizzes.

**Q: Can I use the app on mobile devices?**
A: Yes, the app is fully responsive and works on mobile devices, tablets, and desktops.

**Q: How do I update my profile information?**
A: Currently, profile updates are not available. Contact support for assistance.

### Admin Questions

**Q: How do I become an admin?**
A: Admin privileges are granted by the system administrator. Contact support for admin access.

**Q: Can I see who took my quizzes?**
A: Yes, you can see quiz attempts and user performance in the admin dashboard.

**Q: How many questions can I add to a quiz?**
A: There's no strict limit, but we recommend keeping quizzes manageable (10-50 questions).

**Q: Can I import questions from other sources?**
A: Currently, questions must be entered manually. Bulk import features may be added in future updates.
